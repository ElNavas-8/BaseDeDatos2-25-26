# Tipos de bases de datos NoSQL

### Objetivo de la clase

Conocer los principales tipos de bases de datos NoSQL, entender cómo organizan la información y en qué escenarios resulta más adecuado utilizar cada una.

El propósito es desarrollar criterio técnico para elegir la tecnología más apropiada según el tipo de aplicación y sus necesidades.

### Clasificación general de NoSQL

Las bases de datos NoSQL no siguen un único modelo de organización de datos. A diferencia del mundo relacional, donde predominan las tablas, en NoSQL existen varias formas de estructurar la información.

Las bases de datos NoSQL suelen agruparse en cuatro grandes categorías:

* Clave–valor
* Columnar
* Documentales
* Grafos

En los últimos años se ha añadido un quinto tipo enfocado en inteligencia artificial:

* Bases de datos vectoriales

Cada tipo de base de datos fue diseñado con prioridades distintas. Algunas se enfocan en:

* La velocidad.
* La flexibilidad del esquema.
* La capacidad de manejar relaciones complejas.
* La similitud semántica entre datos.
* Comprender estas diferencias permite evitar decisiones tecnológicas inadecuadas.
